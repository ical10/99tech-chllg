# Token Swap Interface (React + TypeScript + Vite)

## Introduction

This token swap form lets you:
- Pick a token to swap from on the dropdown (with search enabled),
- input the desired amount,
- pick another token to swap to on another search-enabled dropdown,
- swap amount calculated automatically,
- click on Swap button,
- and watch the process unfold!


## Features 

- Searchable token dropdown
- Amount input
- Automatic swap price conversion
- Toaster as a tx progress tracker
